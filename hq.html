About this dashboard

This page reads hq-status.json on load and re-polls it on the interval declared in the file. To point it at real-time data, have each Hermes agent on the VPS write its status to a heartbeat file/endpoint and set this page's STATUS_URL to that endpoint — no other changes needed. The banner flips from config to live automatically when the feed's source field is no longer static-config. Full wiring guide: HQ_MAKE_IT_LIVE.md.

Keep it internal. This page is noindex and isn't linked from the public site. Before relying on it day-to-day, gate it behind a Netlify password (Site settings → Access control → Password protection) or move it to a non-guessable path.
